== Description ==
This plugin inserts a Pink51 conversion tracking tag into the "Thank You" page and Pink51 referral tracking code to footer of all pages of a WooCommerce shop. The plugin has been tested with WordPress 3.6, WooCommerce 2.0.13 and the WooCommerce theme Wootique 1.6.7, though the plugin should work with all WooCommerce themes (tested themes are: Canvas, Wootique, Avada by ThemeFusion).

== Installation ==
1. Unzip and upload the "pink51_woocommerce" plugin directory into your /wp-content/plugins/ directory
2. Activate the plugin through the "Plugins" menu in WordPress
3. In the WordPress admin panel go to settings and then into the "Pink51 Plugin Menu" and enter the Merchant ID provided by Pink51.
